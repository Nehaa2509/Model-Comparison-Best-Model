# Task 14: Model Comparison & Best Model Selection

## Objective
To compare multiple machine learning models and select the best performing model using the Titanic dataset.

## Dataset
Titanic Dataset

## Models Compared
- Logistic Regression
- Decision Tree
- Random Forest
- Support Vector Machine (SVM)

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score

## Best Model
Random Forest was selected based on strong F1-score and good generalization performance.

## Outcome
This task provided hands-on experience in model comparison and algorithm selection similar to industry ML workflows.
